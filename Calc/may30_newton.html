Newton's Method · Ch.13.4
Calc A2 Final23 June 2026
Core Formula
\[\mathbf{x}_{n+1} = \mathbf{x}_n - [DF(\mathbf{x}_n)]^{-1}\,F(\mathbf{x}_n)\]
For \(2\times2\): \(\begin{pmatrix}a&b\\c&d\end{pmatrix}^{-1}=\frac{1}{ad-bc}\begin{pmatrix}d&-b\\-c&a\end{pmatrix}\)
C–1 Two-Step Iteration & Convergence Check HARDER 12 pts
We want to solve \(F(x,y)=\mathbf{0}\) where
\[F(x,y)=\begin{pmatrix}x^2-2y\\xy-1\end{pmatrix}.\] Use initial guess \((x_0,y_0)=(2,\,1)\).
  1. (a)Compute the Jacobian \([DF(x,y)]\) in general, then evaluate at \((2,1)\). Explain why \(F\) is differentiable everywhere.3 pts
  2. (b)Evaluate \(F(2,1)\) and use linear approximation to estimate \(F(2.01,\,1.01)\).2 pts
  3. (c)Perform one Newton step to find \(\mathbf{x}_1\). Compute \(F(\mathbf{x}_1)\) and verify it is closer to \(\mathbf{0}\).4 pts
  4. (d)Perform a second Newton step using \(\mathbf{x}_1\) to find \(\mathbf{x}_2\). What do you notice about convergence?3 pts
(a) Jacobian
\[[DF(x,y)]=\begin{pmatrix}2x&-2\\y&x\end{pmatrix},\qquad [DF(2,1)]=\begin{pmatrix}4&-2\\1&2\end{pmatrix}.\] All entries are polynomials, hence continuous — \(F\) differentiable everywhere.
(b) Linear Approximation
\(F(2,1)=\begin{pmatrix}4-2\\2-1\end{pmatrix}=\begin{pmatrix}2\\1\end{pmatrix}\). With \(\mathbf{h}=\begin{pmatrix}0.01\\0.01\end{pmatrix}\):
\[F(2.01,1.01)\approx\begin{pmatrix}2\\1\end{pmatrix}+\begin{pmatrix}4&-2\\1&2\end{pmatrix}\begin{pmatrix}0.01\\0.01\end{pmatrix}=\begin{pmatrix}2.02\\1.03\end{pmatrix}.\]
(c) First Newton Step
\(\det A = 4(2)-(-2)(1) = 10\). \(A^{-1}=\tfrac{1}{10}\begin{pmatrix}2&2\\-1&4\end{pmatrix}.\) \[A^{-1}F(2,1)=\tfrac{1}{10}\begin{pmatrix}2&2\\-1&4\end{pmatrix}\begin{pmatrix}2\\1\end{pmatrix}=\tfrac{1}{10}\begin{pmatrix}6\\2\end{pmatrix}=\begin{pmatrix}0.6\\0.2\end{pmatrix}.\]
\[\mathbf{x}_1=(2,1)-(0.6,0.2)=(1.4,\,0.8).\]
Check: \(F(1.4,0.8)=\begin{pmatrix}1.96-1.6\\1.12-1\end{pmatrix}=\begin{pmatrix}0.36\\0.12\end{pmatrix}\). Much smaller than \(\begin{pmatrix}2\\1\end{pmatrix}\) ✓
(d) Second Newton Step
\([DF(1.4,0.8)]=\begin{pmatrix}2.8&-2\\0.8&1.4\end{pmatrix}\), \(\det=3.92+1.6=5.52\). \[A^{-1}=\tfrac{1}{5.52}\begin{pmatrix}1.4&2\\-0.8&2.8\end{pmatrix},\quad A^{-1}F=\tfrac{1}{5.52}\begin{pmatrix}1.4(0.36)+2(0.12)\\-0.8(0.36)+2.8(0.12)\end{pmatrix}=\tfrac{1}{5.52}\begin{pmatrix}0.744\\0.048\end{pmatrix}.\]
\[\mathbf{x}_2\approx(1.4-0.1348,\;0.8-0.0087)\approx(1.265,\;0.791).\]
True solution: \(x^2=2y\) and \(xy=1\) gives \(x^3=2\Rightarrow x=2^{1/3}\approx1.2599\). Newton's converging rapidly (quadratic convergence).
C–2 3D System — Newton in \(\mathbb{R}^3\) HARD 10 pts
We want to find a point on the intersection of a sphere and two planes. Define \(F:\mathbb{R}^3\to\mathbb{R}^3\):
\[F(x,y,z)=\begin{pmatrix}x^2+y^2+z^2-3\\x+y-2\\y-z\end{pmatrix}.\] Use initial guess \((x_0,y_0,z_0)=(1,1,1)\).
  1. (a)Compute the \(3\times3\) Jacobian \([DF]\) and evaluate at \((1,1,1)\).3 pts
  2. (b)Evaluate \(F(1,1,1)\). Is \((1,1,1)\) already a solution?1 pt
  3. (c)Compute \([DF(1,1,1)]^{-1}\) using row reduction or the adjugate formula, then perform one Newton step to find \(\mathbf{x}_1\).4 pts
  4. (d)Verify that \(F(\mathbf{x}_1)\) is closer to \(\mathbf{0}\) than \(F(1,1,1)\). What are the exact solutions of \(F=\mathbf{0}\)?2 pts
(a) 3×3 Jacobian
\[[DF(x,y,z)]=\begin{pmatrix}2x&2y&2z\\1&1&0\\0&1&-1\end{pmatrix},\qquad [DF(1,1,1)]=\begin{pmatrix}2&2&2\\1&1&0\\0&1&-1\end{pmatrix}.\]
(b) Value at guess
\[F(1,1,1)=\begin{pmatrix}1+1+1-3\\1+1-2\\1-1\end{pmatrix}=\begin{pmatrix}0\\0\\0\end{pmatrix}.\]
Yes — \((1,1,1)\) is already an exact solution! Newton terminates in 0 steps.
(c) Newton Step (degenerate case)
Since \(F(\mathbf{x}_0)=\mathbf{0}\), the update is \(\mathbf{x}_1 = (1,1,1) - [DF]^{-1}\mathbf{0} = (1,1,1)\). No change needed.
(d) All Solutions
From \(y=z\) and \(x+y=2\), we have \(x=2-y, z=y\). Substituting into the sphere: \((2-y)^2+y^2+y^2=3\Rightarrow 3y^2-4y+1=0\Rightarrow (3y-1)(y-1)=0\).
Solutions: \((1,1,1)\) and \((\tfrac{5}{3},\tfrac{1}{3},\tfrac{1}{3})\).
The initial guess happened to be one solution. Try \((2, 0, 0)\) as a starting point to converge to the second solution.
C–3 Inverse Function Theorem — When Does Newton Fail? 6 pts
Consider \(F(x,y)=\begin{pmatrix}x^2+y^2\\2xy\end{pmatrix}\) with initial guess \((0,0)\).
  1. (a)Compute \([DF(0,0)]\). Is it invertible? What does this mean for Newton's method at this point?2 pts
  2. (b)Evaluate \(F(0,0)\). Even though the Jacobian is singular, is \((0,0)\) a solution to \(F=\mathbf{0}\)?1 pt
  3. (c)State the condition in the Inverse Function Theorem that guarantees \(F\) is locally invertible. Is it satisfied at \((0,0)\)? What about at \((1,1)\)?3 pts
(a)
\([DF(x,y)]=\begin{pmatrix}2x&2y\\2y&2x\end{pmatrix}\), so \([DF(0,0)]=\begin{pmatrix}0&0\\0&0\end{pmatrix}\). \(\det=0\) — not invertible. Newton's method cannot proceed from \((0,0)\): the step \(-[DF]^{-1}F\) is undefined.
(b)
\(F(0,0)=\begin{pmatrix}0\\0\end{pmatrix}\), so \((0,0)\) IS a solution — it's a degenerate case where Newton stalls but convergence is already achieved.
(c) IFT Condition
The IFT requires \(\det[DF(\mathbf{x}_0)]\neq 0\). At \((0,0)\): \(\det=0\) — NOT satisfied. At \((1,1)\): \(\det[DF(1,1)]=\det\begin{pmatrix}2&2\\2&2\end{pmatrix}=0\) — also not satisfied! In fact \(\det[DF]=4x^2-4y^2\), which vanishes on \(|x|=|y|\).
Away from those lines (e.g. at \((2,1)\): \(\det=4(4-1)=12\neq0\)), the IFT applies and \(F\) is locally invertible.

Change of Basis & Coordinate Maps · Ch.5.4

Key Tools
Change-of-basis matrix from \(\mathcal{B}\) to \(\mathcal{C}\): \(\;P_{\mathcal{C}\leftarrow\mathcal{B}} = \bigl[[\mathbf{b}_1]_{\mathcal{C}}\;\;[\mathbf{b}_2]_{\mathcal{C}}\;\;\cdots\bigr]\)  ·  Coord map: \([\mathbf{v}]_{\mathcal{B}}\) satisfies \(\mathbf{v}=\sum c_i \mathbf{b}_i\)  ·  Inverse: \(P_{\mathcal{B}\leftarrow\mathcal{C}}=(P_{\mathcal{C}\leftarrow\mathcal{B}})^{-1}\)
L–1 Coordinate Vectors & Change-of-Basis Matrix 10 pts
Let \(\mathcal{E}=\{e_1,e_2\}\) be the standard basis for \(\mathbb{R}^2\). Consider two new bases:
\[\mathcal{B}=\left\{\begin{pmatrix}1\\1\end{pmatrix},\begin{pmatrix}1\\-1\end{pmatrix}\right\},\qquad \mathcal{C}=\left\{\begin{pmatrix}2\\1\end{pmatrix},\begin{pmatrix}1\\2\end{pmatrix}\right\}.\]
  1. (a)Find \([\mathbf{v}]_{\mathcal{B}}\) for \(\mathbf{v}=\begin{pmatrix}5\\1\end{pmatrix}\). (Express \(\mathbf{v}\) as a linear combination of \(\mathcal{B}\) vectors.)2 pts
  2. (b)Write down the change-of-basis matrix \(P_{\mathcal{E}\leftarrow\mathcal{B}}\) (columns = basis vectors in standard coords). Then find \(P_{\mathcal{B}\leftarrow\mathcal{E}}=(P_{\mathcal{E}\leftarrow\mathcal{B}})^{-1}\).3 pts
  3. (c)Compute \(P_{\mathcal{E}\leftarrow\mathcal{C}}\) and \(P_{\mathcal{C}\leftarrow\mathcal{E}}\).2 pts
  4. (d)Find the change-of-basis matrix \(P_{\mathcal{C}\leftarrow\mathcal{B}}\) without re-doing all the work from scratch. (Hint: compose two maps you already have.)3 pts
(a) Coordinates in B
Solve \(c_1\begin{pmatrix}1\\1\end{pmatrix}+c_2\begin{pmatrix}1\\-1\end{pmatrix}=\begin{pmatrix}5\\1\end{pmatrix}\): \(c_1+c_2=5,\;c_1-c_2=1\Rightarrow c_1=3,c_2=2\).
\[[\mathbf{v}]_{\mathcal{B}}=\begin{pmatrix}3\\2\end{pmatrix}.\]
(b) Change-of-basis P
\(P_{\mathcal{E}\leftarrow\mathcal{B}}=\begin{pmatrix}1&1\\1&-1\end{pmatrix}\). Inverse: \(\det=-2\),
\[P_{\mathcal{B}\leftarrow\mathcal{E}}=\tfrac{1}{-2}\begin{pmatrix}-1&-1\\-1&1\end{pmatrix}=\begin{pmatrix}1/2&1/2\\1/2&-1/2\end{pmatrix}.\]
Check: \(P_{\mathcal{B}\leftarrow\mathcal{E}}\mathbf{v}=\begin{pmatrix}1/2&1/2\\1/2&-1/2\end{pmatrix}\begin{pmatrix}5\\1\end{pmatrix}=\begin{pmatrix}3\\2\end{pmatrix}\) ✓
(c) P for basis C
\(P_{\mathcal{E}\leftarrow\mathcal{C}}=\begin{pmatrix}2&1\\1&2\end{pmatrix}\), \(\det=3\).
\[P_{\mathcal{C}\leftarrow\mathcal{E}}=\tfrac{1}{3}\begin{pmatrix}2&-1\\-1&2\end{pmatrix}.\]
(d) Composed Change of Basis
\(P_{\mathcal{C}\leftarrow\mathcal{B}}=P_{\mathcal{C}\leftarrow\mathcal{E}}\cdot P_{\mathcal{E}\leftarrow\mathcal{B}}\):
\[P_{\mathcal{C}\leftarrow\mathcal{B}}=\tfrac{1}{3}\begin{pmatrix}2&-1\\-1&2\end{pmatrix}\begin{pmatrix}1&1\\1&-1\end{pmatrix}=\tfrac{1}{3}\begin{pmatrix}1&3\\1&-3\end{pmatrix}=\begin{pmatrix}1/3&1\\1/3&-1\end{pmatrix}.\]
L–2 Matrix of a Linear Map in Two Bases HARDER 12 pts
Let \(T:\mathbb{R}^2\to\mathbb{R}^2\) be the linear map \(T\begin{pmatrix}x\\y\end{pmatrix}=\begin{pmatrix}3x-y\\x+y\end{pmatrix}\). Let \(\mathcal{B}=\left\{\begin{pmatrix}1\\1\end{pmatrix},\begin{pmatrix}1\\-1\end{pmatrix}\right\}\) from above.
  1. (a)Write the matrix \([T]_{\mathcal{E}}\) (the standard matrix of \(T\)).1 pt
  2. (b)Compute \(T(\mathbf{b}_1)\) and \(T(\mathbf{b}_2)\) for the two basis vectors, then express each result in \(\mathcal{B}\)-coordinates. Write down \([T]_{\mathcal{B}}\).5 pts
  3. (c)Verify the change-of-basis formula: \([T]_{\mathcal{B}}=P_{\mathcal{B}\leftarrow\mathcal{E}}\,[T]_{\mathcal{E}}\,P_{\mathcal{E}\leftarrow\mathcal{B}}\).4 pts
  4. (d)For \(\mathbf{v}=\begin{pmatrix}5\\1\end{pmatrix}\), compute \([T(\mathbf{v})]_{\mathcal{B}}\) using \(\mathcal{B}\)-coordinates directly (i.e. using \([T]_{\mathcal{B}}\cdot[\mathbf{v}]_{\mathcal{B}}\)) and verify against \([T(\mathbf{v})]_{\mathcal{E}}\) converted to \(\mathcal{B}\).2 pts
(a) Standard Matrix
\[[T]_{\mathcal{E}}=\begin{pmatrix}3&-1\\1&1\end{pmatrix}.\]
(b) Matrix in B
\(T(\mathbf{b}_1)=T\begin{pmatrix}1\\1\end{pmatrix}=\begin{pmatrix}2\\2\end{pmatrix}=2\mathbf{b}_1+0\mathbf{b}_2\Rightarrow [T\mathbf{b}_1]_\mathcal{B}=\begin{pmatrix}2\\0\end{pmatrix}\). \(T(\mathbf{b}_2)=T\begin{pmatrix}1\\-1\end{pmatrix}=\begin{pmatrix}4\\0\end{pmatrix}\). Solving: \(c_1+c_2=4,c_1-c_2=0\Rightarrow c_1=2,c_2=2\Rightarrow[T\mathbf{b}_2]_\mathcal{B}=\begin{pmatrix}2\\2\end{pmatrix}\).
\[[T]_{\mathcal{B}}=\begin{pmatrix}2&2\\0&2\end{pmatrix}.\]
(c) Verify Formula
Using results from L–1: \(P_{\mathcal{B}\leftarrow\mathcal{E}}=\begin{pmatrix}1/2&1/2\\1/2&-1/2\end{pmatrix}\), \(P_{\mathcal{E}\leftarrow\mathcal{B}}=\begin{pmatrix}1&1\\1&-1\end{pmatrix}\). \[P_{\mathcal{B}\leftarrow\mathcal{E}}[T]_\mathcal{E}P_{\mathcal{E}\leftarrow\mathcal{B}}=\begin{pmatrix}1/2&1/2\\1/2&-1/2\end{pmatrix}\begin{pmatrix}3&-1\\1&1\end{pmatrix}\begin{pmatrix}1&1\\1&-1\end{pmatrix}.\] First product: \(\begin{pmatrix}2&0\\1&-1\end{pmatrix}\). Second: \(\begin{pmatrix}2&0\\1&-1\end{pmatrix}\begin{pmatrix}1&1\\1&-1\end{pmatrix}=\begin{pmatrix}2&2\\0&2\end{pmatrix}\) ✓
(d) Apply in B-coords
\([\mathbf{v}]_\mathcal{B}=\begin{pmatrix}3\\2\end{pmatrix}\) (from L-1a). \([T]_\mathcal{B}[\mathbf{v}]_\mathcal{B}=\begin{pmatrix}2&2\\0&2\end{pmatrix}\begin{pmatrix}3\\2\end{pmatrix}=\begin{pmatrix}10\\4\end{pmatrix}\). Direct: \(T\mathbf{v}=\begin{pmatrix}3(5)-1\\5+1\end{pmatrix}=\begin{pmatrix}14\\6\end{pmatrix}\). Convert: \(P_{\mathcal{B}\leftarrow\mathcal{E}}\begin{pmatrix}14\\6\end{pmatrix}=\begin{pmatrix}10\\4\end{pmatrix}\) ✓
L–3 Polynomial Space — Change of Basis in \(\mathcal{P}_2\) HARD 10 pts
Work in \(\mathcal{P}_2\), the space of polynomials of degree \(\leq 2\). Consider two bases:
\[\mathcal{S}=\{1,\,x,\,x^2\}\quad\text{(standard)},\qquad \mathcal{L}=\{1,\,x-1,\,(x-1)^2\}\quad\text{(shifted)}.\]
  1. (a)Express each vector in \(\mathcal{L}\) in terms of \(\mathcal{S}\)-coordinates. Write down \(P_{\mathcal{S}\leftarrow\mathcal{L}}\).3 pts
  2. (b)Find \(P_{\mathcal{L}\leftarrow\mathcal{S}}=(P_{\mathcal{S}\leftarrow\mathcal{L}})^{-1}\) by row-reducing \([P_{\mathcal{S}\leftarrow\mathcal{L}}\;|\;I]\).3 pts
  3. (c)Express \(p(x)=x^2+3x-2\) in \(\mathcal{L}\)-coordinates using \(P_{\mathcal{L}\leftarrow\mathcal{S}}\). Then verify by expanding.4 pts
(a)
\(1=[1,0,0]_\mathcal{S}\), \(x-1=[-1,1,0]_\mathcal{S}\), \((x-1)^2=x^2-2x+1=[1,-2,1]_\mathcal{S}\).
\[P_{\mathcal{S}\leftarrow\mathcal{L}}=\begin{pmatrix}1&-1&1\\0&1&-2\\0&0&1\end{pmatrix}.\]
(b) Row reduction for inverse
Row-reduce \([P|\,I]\): \(P\) is upper triangular with all 1s on diagonal — easily inverted by back substitution.
\[P_{\mathcal{L}\leftarrow\mathcal{S}}=\begin{pmatrix}1&1&1\\0&1&2\\0&0&1\end{pmatrix}.\]
(Verify: \(P_{\mathcal{S}\leftarrow\mathcal{L}}P_{\mathcal{L}\leftarrow\mathcal{S}}=I\) ✓)
(c) Express p(x) in L-coords
\([p]_\mathcal{S}=\begin{pmatrix}-2\\3\\1\end{pmatrix}\). Then: \[[p]_\mathcal{L}=P_{\mathcal{L}\leftarrow\mathcal{S}}\begin{pmatrix}-2\\3\\1\end{pmatrix}=\begin{pmatrix}1&1&1\\0&1&2\\0&0&1\end{pmatrix}\begin{pmatrix}-2\\3\\1\end{pmatrix}=\begin{pmatrix}2\\5\\1\end{pmatrix}.\]
\[p(x)=2\cdot1+5\cdot(x-1)+1\cdot(x-1)^2.\]
Verify: \(2+5(x-1)+(x-1)^2=2+5x-5+x^2-2x+1=x^2+3x-2\) ✓

    int arr[5] = {10, 20, 30, 40, 50};

P–2 Dynamic Memory — Spot the Bugs HARDER 10 pts
The following code has multiple memory-related bugs. Identify every bug, name it, and write the corrected version. 
// Intended: create array of n ints, double each element, print, free
void processArray(int n) {
    int* arr = new int[n];

    for (int i = 0; i <= n; i++)   // Bug?
        arr[i] = i * 2;

    int* p = arr;
    delete p;                         // Bug?
    arr = nullptr;

    for (int i = 0; i < n; i++)
        std::cout << arr[i] << " ";   // Bug?
}

void leakyFunction() {
    int* x = new int(42);
    int* y = new int(100);
    x = y;                            // Bug?
    delete x;
}
  1. (a)In processArray: what is the bug on the loop condition line (i <= n)? What type of error is it?2 pts
  2. (b)What is wrong with delete p? What should it be?2 pts
  3. (c)After arr = nullptr, the loop prints arr[i]. What is this bug called? What happens at runtime?2 pts
  4. (d)In leakyFunction: the line x = y causes a memory leak. Explain why, and write the corrected function that properly frees all memory.4 pts
(a) Off-by-one
i <= n allows i = n, writing to arr[n] which is out-of-bounds. This is a buffer overflow — undefined behaviour, possible crash or data corruption. Fix: i < n.
(b) Wrong delete form
arr was allocated with new int[n] (array form). Freeing with delete p (scalar form) is undefined behaviour — must use delete[] p or delete[] arr.
Fix: delete[] p;
(c) Use-after-free / null dereference
After arr = nullptr, dereferencing arr[i] is a null pointer dereference — undefined behaviour, usually a segfault. The print loop must happen BEFORE the delete and null assignment.
(d) Memory leak fix
When x = y runs, x no longer points to the original new int(42) allocation. That memory becomes unreachable — memory leak. Only one of the two allocations gets freed (delete x frees what y pointed to). Fix: 
void leakyFunction() {
    int* x = new int(42);
    int* y = new int(100);
    delete x;   // free original x before overwriting pointer
    x = y;
    delete x;   // now frees what y pointed to
    // y is now a dangling pointer — don't use it after this
}
P–3 Write from Memory — Dynamic 2D Array HARD 12 pts
Without referencing any notes, write complete C++ code for each part below.
  1. (a)Write a function int** createMatrix(int rows, int cols) that allocates a 2D array of ints on the heap using pointer-to-pointer, initialises every element to 0, and returns the pointer.4 pts
  2. (b)Write a corresponding function void deleteMatrix(int** mat, int rows) that correctly frees all allocated memory without leaks.3 pts
  3. (c)Write a function void fillDiagonal(int** mat, int n, int val) that sets mat[i][i] = val for all i in an \(n\times n\) matrix.2 pts
  4. (d)In main, use your three functions to create a \(4\times4\) matrix, fill its diagonal with 7, print the full matrix, then free it cleanly. The output should look like a grid.3 pts
(a) createMatrix
int** createMatrix(int rows, int cols) {
    int** mat = new int*[rows];
    for (int i = 0; i < rows; i++) {
        mat[i] = new int[cols];
        for (int j = 0; j < cols; j++)
            mat[i][j] = 0;
    }
    return mat;
}
(b) deleteMatrix
void deleteMatrix(int** mat, int rows) {
    for (int i = 0; i < rows; i++)
        delete[] mat[i];   // free each row first
    delete[] mat;         // then free the array of pointers
}
(c) fillDiagonal
void fillDiagonal(int** mat, int n, int val) {
    for (int i = 0; i < n; i++)
        mat[i][i] = val;
}
(d) main
int main() {
    int n = 4;
    int** m = createMatrix(n, n);
    fillDiagonal(m, n, 7);
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++)
            std::cout << m[i][j] << " ";
        std::cout << "\n";
    }
    deleteMatrix(m, n);
    return 0;
}
// Expected output:
// 7 0 0 0
// 0 7 0 0
// 0 0 7 0
// 0 0 0 7
Remember: always delete inner arrays before the outer pointer array. Reversing the order leaves the inner allocations leaked.
