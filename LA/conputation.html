Chapter 3 · Computational

Gaussian Elimination Recipes

Time2–4 min per 3×3
Recipe · 01 Row-reduce a matrix to RREF
1. Find the leftmost non-zero column. If the top entry is zero, swap rows. 2. Scale that row so the pivot is \(1\). 3. Use row 3 ops to clear every other entry in that column (both above and below). 4. Cover the pivot row, recurse on the submatrix below-and-right. 5. Stop when the submatrix is all zeros.
Cross-check: count pivots — that's the rank.
Recipe · 02 Read off solutions to \(A\mathbf x=\mathbf b\)
Augment to \([A\,|\,\mathbf b]\) and reduce. 1. If any row reads \([0\cdots0\,|\,c\ne0]\): inconsistent, no solution. 2. Otherwise set free variables to \(0\) → read off the particular solution \(\mathbf x_p\) directly from the pivots. 3. For each free variable in turn, set it to 1 (others to 0) → gives a basis vector of \(\operatorname{Null}(A)\). 4. Full solution: \(\mathbf x_p+\operatorname{span}\{\text{null-space basis}\}\).
Apply · 03 Test invertibility of a square \(A\)
Reduce \(A\) to RREF. Invertible \(\iff\) RREF \(=I\) \(\iff\) \(n\) pivots \(\iff\) \(\det(A)\ne0\) \(\iff\) \(\operatorname{Null}(A)=\{0\}\). To compute \(A^{-1}\) directly: row-reduce \([A\,|\,I]\) to \([I\,|\,A^{-1}]\).
Chapter 4 · Computational

Matrix Algebra & LU

Time5–8 min for a 3×3 LU
Recipe · 01 LU decomposition by hand (no pivoting)
1. Run elimination using only "\(R_i\to R_i-\ell_{ij}R_j\)" (no swaps, no scaling). 2. Track every multiplier \(\ell_{ij}\); place it in row \(i\), col \(j\) of \(L\). 3. The resulting echelon matrix is \(U\); fill \(L\)'s diagonal with \(1\)s. 4. Verify: \(LU=A\).
If any pivot lands on zero, a swap is required → see PLU below.
Recipe · 02 PLU when pivoting is needed
1. Whenever a pivot is zero, swap with a lower row whose entry is non-zero. 2. Record the swap as a permutation matrix \(P\) (each swap left-multiplies the previous \(P\)). 3. Continue elimination; the multipliers fill \(L\) using the post-swap row indices. 4. Result: \(PA=LU\). Verify by computing both sides.
Apply · 03 Solve \(A\mathbf x=\mathbf b\) via LU
Once \(A=LU\) (or \(PA=LU\)) is cached, every new \(\mathbf b\) costs only \(\sim 2n^2\) work: 1. (If using \(P\): replace \(\mathbf b\) with \(P\mathbf b\).) 2. Forward substitution: solve \(L\mathbf y=\mathbf b\) top-down. 3. Back substitution: solve \(U\mathbf x=\mathbf y\) bottom-up. 4. Verify: \(A\mathbf x=\mathbf b\).
Recipe · 04 Block matrix multiplication
Partition \(A=\begin{psmallmatrix}A_{11}&A_{12}\\A_{21}&A_{22}\end{psmallmatrix}\), \(B=\begin{psmallmatrix}B_{11}&B_{12}\\B_{21}&B_{22}\end{psmallmatrix}\) with compatible block sizes. Then \(AB\)'s blocks are computed by the usual matrix-product formula treating the blocks as scalars: \((AB)_{11}=A_{11}B_{11}+A_{12}B_{21}\), and so on.
Trick: If \(A\) is block upper triangular and \(B\) is block lower triangular, many blocks vanish.
Chapter 5 · Computational

Bases, Coordinates, Rank

Time5–10 min per problem
Recipe · 01 Find a basis for a span
Given vectors \(v_1,\dots,v_k\): 1. Place them as rows of a matrix \(M\). 2. Row-reduce \(M\) to RREF. 3. The non-zero rows of RREF form a basis of \(\operatorname{span}\{v_i\}\). The number of non-zero rows is the dimension.
Alternative (column method): place as columns, reduce, and take the original columns in the pivot positions.
Recipe · 02 Compute the coordinate vector \([v]_{\mathcal B}\)
Given a basis \(\mathcal B=(b_1,\dots,b_n)\) and a vector \(v\): 1. Set up \(v=c_1b_1+\cdots+c_nb_n\) as a linear system \(Bc=v\), where \(B\) has the \(b_j\) as columns. 2. Solve via Gaussian elimination (or, if \(B\) is invertible, use \(c=B^{-1}v\)). 3. \([v]_{\mathcal B}=(c_1,\dots,c_n)^T\).
For abstract spaces: first convert to a standard basis representation (e.g. for \(P_2\), use \((1,x,x^2)\)).
Recipe · 03 Build the change-of-basis matrix \(I_{\mathcal C\leftarrow\mathcal B}\)
Columns of \(I_{\mathcal C\leftarrow\mathcal B}\) are the \(\mathcal B\)-basis vectors written in \(\mathcal C\)-coordinates.
1. For each \(b_j\), solve \(b_j=\sum_i [b_j]_{\mathcal C,i}\,c_i\) to find the \(\mathcal C\)-coordinates. 2. Stack these as columns.
Sanity check: \(I_{\mathcal C\leftarrow\mathcal B}\cdot I_{\mathcal B\leftarrow\mathcal C}=I\). One direction is often easier than the other — compute the easier one, invert for the other.
Apply · 04 Change the matrix of a linear map
Given \(T:V\to V\) and \([T]_{\mathcal B}\), to get \([T]_{\mathcal C}\): 1. Compute \(P=I_{\mathcal C\leftarrow\mathcal B}\) and \(P^{-1}=I_{\mathcal B\leftarrow\mathcal C}\). 2. \([T]_{\mathcal C}=P\,[T]_{\mathcal B}\,P^{-1}\). 3. Cross-check directly: the \(j\)-th column of \([T]_{\mathcal C}\) is \([T(c_j)]_{\mathcal C}\).
Invariants check: trace, determinant, eigenvalues, and characteristic polynomial must agree between \([T]_{\mathcal B}\) and \([T]_{\mathcal C}\).
Recipe · 05 Find the four fundamental subspaces
Reduce \(A\) to RREF. Then: Col(A): original columns of \(A\) in the pivot positions. Row(A): non-zero rows of RREF. Null(A): set free variables to standard basis vectors, back-substitute for pivots. LeftNull(A): reduce \(A^T\) and apply Null procedure (or: find row dependencies in original \(A\)). Dimensions: rank \(=\dim\operatorname{Col}=\dim\operatorname{Row}\); nullity \(=n-r\); leftnullity \(=m-r\).
Chapter 6 · Computational

Inner Products, Cholesky, Gram-Schmidt, Projections

Time10–15 min for full QR
This is the chapter the professor emphasises most. Yang's specific column-shearing perspective of Gram-Schmidt (the "dot product with itself and with every other column" procedure) is below in Recipe 03 — the one you asked about.
Recipe · 01 Gram matrix and positive-definiteness check
Given \(A\), compute \(G=A^TA\). Then \(G_{ij}=\mathbf a_i\cdot\mathbf a_j\) (dot product of columns of \(A\)). Always symmetric PSD. Test PD: compute its leading principal minors — all must be positive (Sylvester's criterion), or equivalently all eigenvalues \(>0\), or Cholesky succeeds.
Recipe · 02 Cholesky decomposition \(G=LL^T\)
For a positive definite \(G\), compute \(L\) lower triangular with positive diagonal: 1. \(l_{11}=\sqrt{G_{11}}\). 2. \(l_{i1}=G_{i1}/l_{11}\) for \(i>1\) (first column of \(L\)). 3. For \(j=2,\dots,n\): \(l_{jj}=\sqrt{G_{jj}-\sum_{kj\). 4. If any \(l_{jj}^2\le0\), \(G\) is not positive definite — Cholesky has detected non-PDness.
★ Apply · 03 Gram-Schmidt via column-shearing (Prof. Yang's procedure → QR)
This is the "dot product with itself and other columns" recipe from §6.9.2. Given \(A\) with independent columns \([\mathbf a_1\,|\,\mathbf a_2\,|\,\cdots\,|\,\mathbf a_n]\):
1. Compute \(\mathbf a_1\cdot\mathbf a_1\) (= \(\|\mathbf a_1\|^2\)) and the dot products \(\mathbf a_1\cdot\mathbf a_j\) for every \(j>1\). 2. The ratios \(r_{1j}=\dfrac{\mathbf a_1\cdot\mathbf a_j}{\mathbf a_1\cdot\mathbf a_1}\) are the shearing coefficients. They become row 1 of an upper triangular matrix \(U\) to the right. 3. Shear: replace \(\mathbf a_j\to\mathbf a_j-r_{1j}\mathbf a_1\) for each \(j>1\). Now all columns \(j>1\) are orthogonal to column 1. 4. Move on to column 2 and repeat steps 1–3 with the second column on columns 3 onward — these ratios become row 2 of \(U\). 5. Continue until you've used every column. You now have an orthogonal matrix \(W=[\mathbf w_1\,|\,\cdots\,|\,\mathbf w_n]\) and \(A=WU\). 6. Normalise: let \(D=\operatorname{diag}(\|\mathbf w_i\|)\), then \(Q=WD^{-1}\) is orthonormal and \(R=DU\) is upper triangular with positive diagonal. 7. Final QR: \(A=QR\).

Why this works: each shearing is a column operation \(\mathbf a_j\to\mathbf a_j-r_{ij}\mathbf a_i\), which is exactly right-multiplication by an elementary upper-triangular matrix. Compose them all and you get the upper-triangular \(U\); normalise to get \(R\).
Recipe · 04 Express a vector in an ONB
Given an ONB \((q_1,\dots,q_n)\): the coordinates of \(v\) are just inner products: \[ [v]_i = \langle v,q_i\rangle,\qquad v=\sum_i \langle v,q_i\rangle\,q_i.\] No system to solve — orthonormality means \(Q^TQ=I\), so \(Q^{-1}=Q^T\).
Recipe · 05 Orthogonal projection & least squares
Project a vector \(\mathbf b\) onto \(\operatorname{Col}(A)\): 1. Form the projection matrix \(P=A(A^TA)^{-1}A^T\). Check \(P^2=P\) and \(P^T=P\). 2. Projection of \(\mathbf b\) is \(\hat{\mathbf b}=P\mathbf b\). 3. Residual: \(\mathbf r=\mathbf b-\hat{\mathbf b}\); verify \(\mathbf r\perp\operatorname{Col}(A)\) (i.e., \(A^T\mathbf r=\mathbf 0\)).
Least squares to (possibly inconsistent) \(A\mathbf x=\mathbf b\): solve the normal equations \(A^TA\mathbf x=A^T\mathbf b\). With \(A=QR\): just solve \(R\mathbf x=Q^T\mathbf b\) by back-substitution — far more efficient.
Apply · 06 Line / plane / polynomial fitting
Fit \(y=mx+c\) to data \((x_i,y_i)\): set \(A=\begin{psmallmatrix}x_1&1\\x_2&1\\\vdots&\vdots\end{psmallmatrix}\), \(\mathbf b=(y_1,y_2,\ldots)^T\); solve \(A^TA(m,c)^T=A^T\mathbf b\). For a degree-\(d\) polynomial fit: same idea with \(A_{ij}=x_i^{j-1}\) (Vandermonde-style).
Chapter 7 · Computational

Determinants

Time1–3 min for 3×3
Recipe · 01 Cofactor expansion (\(n\times n\))
Pick a row or column with many zeros. For each non-zero entry \(a_{ij}\): 1. Delete row \(i\) and column \(j\) — leaves a \((n-1)\times(n-1)\) minor \(M_{ij}\). 2. Sign: \((-1)^{i+j}\) (checkerboard). 3. \(\det(A)=\sum_{j}(-1)^{i+j}a_{ij}\det(M_{ij})\). 4. Recurse until you hit \(2\times2\): \(\det\!\begin{psmallmatrix}a&b\\c&d\end{psmallmatrix}=ad-bc\).
Recipe · 02 Determinant via row reduction
Faster for large matrices. Row-reduce \(A\) to upper triangular \(U\), tracking: Swaps: each one multiplies the determinant by \(-1\). Scaling row by \(c\): divides the determinant by \(c\) (so the new matrix has \(c\) times the original \(\det\)). "Add multiple of one row to another": doesn't change \(\det\). End: \(\det(A)=\pm\prod\) diagonal entries of \(U\), with sign from the number of swaps.
Apply · 03 Cramer's rule & properties
For invertible \(A\), solve \(A\mathbf x=\mathbf b\) by \(x_i=\det(A_i)/\det(A)\), where \(A_i\) is \(A\) with column \(i\) replaced by \(\mathbf b\). Mostly for theory — slow in practice.
Key properties to spot-check: \(\det(I)=1\); \(\det(AB)=\det(A)\det(B)\); \(\det(A^T)=\det(A)\); \(\det(A^{-1})=1/\det(A)\); \(\det(cA)=c^n\det(A)\).
Chapter 8 · Computational

Eigenstuff, Spectral, SVD

Time15–25 min for full SVD
Recipe · 01 Find eigenvalues & eigenvectors
1. Compute \(p_A(\lambda)=\det(\lambda I-A)\) — the characteristic polynomial. 2. Factor \(p_A\) to find the eigenvalues \(\lambda_i\) (over \(\mathbb C\) if needed). 3. For each \(\lambda_i\): solve \((A-\lambda_iI)\mathbf v=\mathbf 0\) by row reduction → basis vectors of the eigenspace \(E_{\lambda_i}\). 4. The dimension of each \(E_{\lambda_i}\) is the geometric multiplicity; the multiplicity of \(\lambda_i\) as a root is the algebraic multiplicity.
Recipe · 02 Diagonalise: \(A=PDP^{-1}\)
Requires \(n\) independent eigenvectors (i.e., every algebraic multiplicity equals geometric). 1. Find eigenvalues \(\lambda_1,\dots,\lambda_n\) (with multiplicity) and eigenvectors \(\mathbf v_1,\dots,\mathbf v_n\). 2. Set \(P=[\mathbf v_1\,|\,\cdots\,|\,\mathbf v_n]\) and \(D=\operatorname{diag}(\lambda_1,\dots,\lambda_n)\) in the same column order. 3. Then \(A=PDP^{-1}\), \(A^k=PD^kP^{-1}\), \(e^A=Pe^DP^{-1}\) etc.
If not diagonalisable: sketch the Schur form \(A=QTQ^*\) (upper triangular) instead.
Apply · 03 \(A^k\), Markov chains, linear ODEs
Once \(A=PDP^{-1}\): \(A^k=PD^kP^{-1}\) is just raising diagonal entries to the \(k\). Computes powers in \(O(n^3)\) regardless of \(k\). Markov chain steady state: the eigenvector for \(\lambda=1\), normalised so its entries sum to \(1\); all other eigenvalues have \(|\lambda|<1\) so their contributions decay. Linear ODE \(\mathbf x'=A\mathbf x\): solution is \(\mathbf x(t)=Pe^{Dt}P^{-1}\mathbf x_0\), where \(e^{Dt}=\operatorname{diag}(e^{\lambda_i t})\). Stability classified by signs of \(\operatorname{Re}(\lambda_i)\).
Recipe · 04 Spectral decomposition (symmetric matrices)
For symmetric \(A=A^T\): 1. Find eigenvalues — guaranteed all real. 2. Find eigenvectors for each. 3. Eigenvectors from distinct eigenvalues are automatically orthogonal. Within a repeated eigenvalue, apply Gram-Schmidt to the eigenspace basis. 4. Normalise — assemble \(Q\), an orthogonal matrix. Set \(\Lambda=\operatorname{diag}(\lambda_i)\). 5. Then \(A=Q\Lambda Q^T\); since \(Q\) is orthogonal, \(Q^{-1}=Q^T\) — no inversion needed.
★ Apply · 05 Singular Value Decomposition step-by-step
For any \(m\times n\) matrix \(A\), compute \(A=U\Sigma V^T\): 1. Form \(A^TA\) (\(n\times n\), symmetric PSD). 2. Find its eigenvalues \(\lambda_1\ge\lambda_2\ge\cdots\ge\lambda_n\ge 0\). The singular values are \(\sigma_i=\sqrt{\lambda_i}\) (only the non-zero ones count toward rank). 3. Find orthonormal eigenvectors of \(A^TA\) — these become the columns of \(V\), in the same order as the \(\sigma_i\). 4. For each \(\sigma_i>0\): set \(\mathbf u_i=A\mathbf v_i/\sigma_i\) — these are orthonormal columns of \(U\). 5. If \(m>r=\operatorname{rank}(A)\): extend \(\{\mathbf u_1,\dots,\mathbf u_r\}\) to an ONB of \(\mathbb R^m\) (Gram-Schmidt on any basis of the complement). 6. Assemble \(\Sigma\) as an \(m\times n\) matrix with the \(\sigma_i\) on the diagonal and zeros elsewhere. 7. Verify: \(U\Sigma V^T=A\).
Why the procedure works: \(A^TA = V\Sigma^TU^TU\Sigma V^T = V\Sigma^T\Sigma V^T\), and \(\Sigma^T\Sigma\) is diagonal with the \(\sigma_i^2\) on the diagonal — exactly the spectral decomposition of \(A^TA\).
Apply · 06 Low-rank approximation via SVD
Given \(A=U\Sigma V^T\) with singular values \(\sigma_1\ge\cdots\ge\sigma_r>0\), the best rank-\(k\) approximation is \[ A_k = \sum_{i=1}^k \sigma_i\,\mathbf u_i\,\mathbf v_i^T \] — keep the top \(k\) terms of the outer-product expansion. Error in Frobenius norm: \(\|A-A_k\|_F=\sqrt{\sum_{i>k}\sigma_i^2}\). Used in PCA, image compression, recommender systems.
Recipe · 07 Classify a quadratic form
Given \(q(\mathbf x)=\mathbf x^TA\mathbf x\) (symmetrise \(A\) if needed: \(A\leftarrow(A+A^T)/2\)): 1. Compute the eigenvalues of \(A\). 2. Classify by sign pattern: all \(>0\) → positive definite (ellipsoid); all \(<0\) → negative definite; mix of \(\pm\) → indefinite (saddle / hyperboloid); some zero → degenerate (paraboloid, cylinder, …). 3. For sketching the surface: diagonalise via the spectral theorem to find principal axes.
Quick test (Sylvester): PD \(\iff\) all leading principal minors \(>0\).
